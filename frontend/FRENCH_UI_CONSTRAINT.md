# CONSTRAINT: All UI text must be in French

Every user-visible string in the frontend must be in French:
- Button labels
- Form labels and placeholders
- Headings and subtitles
- Empty state messages
- Toast notifications (success, error, info)
- Loading text
- Error messages
- Navigation links
- Badge labels
- Tooltips

Code, variable names, comments, and PropTypes stay in English.

Examples:
- "Submit" → "Envoyer"
- "Loading..." → "Chargement..."
- "No results found" → "Aucun résultat trouvé"
- "Error" → "Erreur"
- "Success" → "Succès"
- "Cancel" → "Annuler"
- "View details" → "Voir le détail"
- "Notify SME" → "Notifier la PME"
- "Run matching" → "Lancer le matching"
- "Register" → "S'inscrire"
- "Publish" → "Publier"
